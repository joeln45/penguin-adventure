# Penguin Adventure

A 2D platformer written in Java. Help a penguin collect stars, dodge enemies, and reach the igloo across multiple levels.

> Originally built as a Stirling University CSCU9N6 coursework project (2024), now being modernized into a portfolio piece — Maven build, refactored architecture, unit tests, CI, and new gameplay features.
